Looking for how a property is distributed within one genome? Use Property Explorer with Grouping set to "species (genome)" -- that view isn't duplicated here to avoid showing the same plot twice.

Effect Sizes

Rank-biserial correlation, sorted by magnitude. Click a bar to jump to that property in Property Explorer (protein-level properties) or the CDS table in Property Explorer (CDS-level properties).

Sensitivity

Leave-one-subgroup-out: each cell is how much a property's apparent effect size changes when one subgroup is excluded from the comparison. Positive shrinkage means the effect got weaker without that subgroup -- that subgroup was driving it. Negative shrinkage means the effect got stronger without it -- that subgroup had been diluting it. Excluded-subgroup columns that would leave one side of the primary grouping with zero genomes (shrinkage undefined for every property) are omitted rather than shown as a blank stripe.

Codon Usage

Relative codon-usage frequency (% of all codons in that genome), one row per genome (ordered the same way as Species View), one column per codon (alphabetical). This is plain codon frequency, not RSCU.
